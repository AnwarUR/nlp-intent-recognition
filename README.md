# NLP Intent Recognition Pipeline

A Natural Language Processing pipeline that classifies user intents 
from text data using machine learning techniques.

## What it does
- Scrapes and processes text reviews from web sources
- Cleans and tokenizes text data
- Trains a classification model to detect user intent
- Evaluates model accuracy and performance

## Technologies Used
- Python
- NLTK (Natural Language Toolkit)
- Scikit-Learn
- Pandas
- NumPy

## Project Structure

nlp-intent-recognition/
│
├── data/          # Raw and processed datasets
├── notebooks/     # Jupyter notebooks with analysis
├── src/           # Source code
└── README.md      # Project documentation


## Key Concepts Applied
- Text preprocessing and tokenization
- Feature extraction (TF-IDF vectorization)
- Intent classification model training
- Model evaluation metrics
